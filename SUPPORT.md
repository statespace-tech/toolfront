# Support

Thanks for using Statespace. Here is the best way to get help.

## Bug Reports

If you think you found a bug, please open a GitHub issue using the bug report
template. Include reproduction steps, expected behavior, and logs where
possible.

## Feature Requests

Open a GitHub issue using the feature request template. Explain the problem you
are trying to solve and any alternatives you considered.

## Community Support

Join the community Discord for usage questions and general discussion:

https://discord.gg/rRyM7zkZTf

## Commercial Support

For billing or enterprise support, contact:

sales@statespace.com

## Security Issues

Do not open public issues for security concerns. Please follow the instructions
in `SECURITY.md`.
