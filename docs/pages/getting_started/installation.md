# Installation

ToolFront is available on PyPI as `toolfront` so installation[^1] is as simple as:

=== ":fontawesome-brands-python: &nbsp; pip"

    ```bash
    pip install toolfront
    ```

=== ":simple-uv: &nbsp; uv"

    ```bash
    uv add toolfront
    ```

=== ":simple-poetry: &nbsp; poetry"

    ```bash
    poetry add toolfront
    ```
[^1]: Requires Python 3.10+
