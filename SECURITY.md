# Security Policy

## Supported Versions

We provide security updates for:

- The latest released version
- The `main` branch

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub Security Advisories:

https://github.com/statespace-tech/statespace/security/advisories/new

Include as much detail as possible to help us reproduce and fix the issue:

- A description of the vulnerability and its impact
- Steps to reproduce
- A proof-of-concept (if available)
- Affected versions or commit hashes

We will acknowledge receipt within 5 business days and work with you on a fix
and coordinated disclosure timeline.
